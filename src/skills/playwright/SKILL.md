# Playwright Skill

Use this skill to open, navigate and interact with graphical interfaces of websites, take screenshots, and perform web automations.

## When to Use

Use this skill when:
- You are specifically asked to open, visit websites wuign playwright
- You need to access graphical interface of an website
- You need to execute scripts in a browser
- You need to take screenshots of websites
- You need to perform website automations
- Use browser for JavaScript-heavy sites that WebFetch cannot parse
- Take screenshots for visual evidence in research reports

## Usage

Call the help command first in order to get the list of available commands and their usage.:

```bash
playwright-cli --help
```

Then call desired command eg. to open website:
```bash
playwright-cli open https://example.com
```

## Examples

### Discover
```bash
playwright-cli --help
```

### Open Website
```bash
playwright-cli open https://example.com
```
